## Quick start

1. Install package

    ```sh
    npm i
    ```

2. Start project

    ```sh
    npm start
    ```

3. Access `http://localhost:3000` in your browser.

## Deploy

- Basic deploy:

  ```sh
  npm run build
  ```

- With docker and also docker compose:

  ```sh
  docker compose up -d --build
  ```

Default will run on <http://127.0.0.1:3080>
